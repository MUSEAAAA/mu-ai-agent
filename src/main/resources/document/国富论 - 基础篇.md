# 亚当·斯密《国富论》核心思想总结

## 作者与背景

亚当·斯密（Adam Smith，1723—1790），苏格兰启蒙思想家，现代经济学之父。《国民财富的性质和原因的研究》（简称《国富论》）出版于1776年，是现代经济学的奠基之作。

## 核心理论

### 1. 劳动分工（Division of Labor）

斯密认为劳动分工是提高生产效率的最重要原因。他以制针工厂为例：一个工人独立制针，一天可能做不出20根；但通过分工（抽丝、拉直、切断、磨尖等18道工序），10个工人一天可生产48,000根针，人均效率提高了数百倍。

分工提高效率的三个原因：
- 熟练程度提升（熟能生巧）
- 减少工序转换的时间损失
- 促进机械发明（分工使人专注于单一任务，更容易发现改进方法）

分工的深度受市场范围限制——**市场规模决定分工深度**，这被称为"斯密定理"。

### 2. 看不见的手（Invisible Hand）

斯密最著名的隐喻。每个人在追求自身利益时，仿佛被一只"看不见的手"引导，最终促进了社会公共利益。屠夫、酿酒师、面包师提供晚餐，并非出于仁慈，而是出于自利。

**关键洞察**：在自由市场下，个人自利与社会利益可以实现自发协调，不需要中央指令。

### 3. 价值与价格理论

斯密区分了：
- **使用价值**：物品的效用（水很有用但很便宜）
- **交换价值**：购买其他物品的能力（钻石没什么用但很贵）

"斯密悖论"：为什么使用价值高的东西交换价值低？

斯密提出了劳动价值论的早期版本——劳动是衡量一切商品交换价值的真实尺度。

### 4. 资本积累与经济增长

经济增长的源泉：
- 资本积累（储蓄转化为投资）
- 劳动分工深化
- 市场范围扩大

斯密主张节俭、反对浪费，认为"节俭是财富的直接增加"。

## 政策主张

### 自由放任（Laissez-faire）

斯密主张减少政府干预，认为市场机制能够自我调节。政府的职能应限于：
1. 国防（保护社会免受外敌入侵）
2. 司法（保护每个社会成员免受不公正对待）
3. 公共工程与制度（建立和维护私人无利可图但社会必需的公共设施）

### 税收四原则

1. **公平**：按能力纳税
2. **确定**：税制明确，不能随意变更
3. **便利**：纳税时间与方式应方便纳税人
4. **经济**：征税成本应最小化

## 历史影响

《国富论》奠定了古典政治经济学的基础，影响了李嘉图、马尔萨斯、马克思等后世思想家。它提出的市场机制理论至今仍是市场经济理论的基石。

## 对知识经济的启示

- 知识经济中，智力分工（专业化知识领域）比体力分工更为精细
- 知识的"市场范围"因互联网而无限扩大，知识和人力资本持续深化
- "看不见的手"在知识市场中表现为知识生产者的自利动机最终推动知识进步
