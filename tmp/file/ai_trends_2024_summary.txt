《中国发展报告2024》人工智能趋势核心摘要（国务院发展研究中心）

一、四大技术经济特征
1. 自我增强型颠覆技术：首个能以自我组织、自我增强方式高速迭代的颠覆性技术。
2. 底座型通用技术：大模型显著降低知识生产边际成本；应用成本低、规模效应强、开源开放，驱动多产业“二次创新”。
3. 强垄断性（赢者通吃）：参数量、数据、算力持续扩大 → 功能指数级提升 → 领先者地位难以撼动。
4. 黑箱特性突出：超万亿参数模型难以解释，研发者亦难精准调控生成过程，对治理构成挑战。

二、三大发展趋势
1. 基础设施化：数据/信息/知识密集型行业率先落地，AI将成为新型经济社会基础设施。
2. 技术路线多元化：大模型非唯一路径，具身智能、类脑计算、小模型轻量化等新路线将持续涌现。
3. 前沿技术融合加速：与先进制造、生命科学、量子计算、区块链结合，催生具身智能机器人、类脑智能、精准医疗、智能合约等新赛道。

三、我国发展AI的三大优势
- 市场与场景优势：全球最大数字基建 + 最广阔消费级市场 + 全球门类最全制造业体系；
- 科技与人才优势：AI论文/专利全球第一 + 全球最大高等教育体系（45%理工类毕业生）；
- 智能基建优势：算力总规模全球第二 + 电信/绿电设施全球领先 + 数据资源快速增长。

四、政策建议方向
坚持“发展优先、守好底线、应用驱动、科技引领”原则，聚焦：
→ 强化AI在制造业、民生服务等场景的规模化落地（应用驱动）；
→ 持续投入基础模型、芯片、算法等底层技术攻关（科技引领）。
