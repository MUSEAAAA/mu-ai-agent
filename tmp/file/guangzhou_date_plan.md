# 广州白云区浪漫约会计划

## 约会地点推荐（5公里范围内）

### 1. 麓湖公园 - 浪漫自然之选
- **位置**：广州市白云区，白云山脚下
- **开放时间**：全天开放（聚芳园6:00-18:00，鸿鹄楼8:00-18:00）
- **门票**：免费
- **特色体验**：
  - 湖面划船（脚踏船/电动船，30元/小时）
  - 落羽杉林漫步（秋冬季变焦糖色）
  - 鸟语林喂鸟（10元/人）
  - 鸿鹄楼登高观景
- **最佳时段**：下午至日落前，光线柔和，湖光山色最美

### 2. 白云山 - 经典地标之选
- **位置**：广州市白云区广园中路801号
- **开放时间**：景区全天开放；索道9:00-18:00；电瓶车9:00-17:00
- **门票**：进山5元/人，摩星岭5元，鸣春谷10元
- **特色体验**：
  - 摩星岭登顶（广州最高点，360°俯瞰城市）
  - 白云晚望看日落
  - 山顶广场夜景（21:00索道下山，欣赏珠江新城灯光秀）
- **最佳时段**：清晨登山看日出，或傍晚登山看日落

### 3. 白云万达广场 - 现代时尚之选
- **位置**：广州市白云区白云新城商圈
- **营业时间**：周一至周日10:00-22:00
- **特色体验**：
  - 顶楼露台夜景（视野超棒，夜晚看城市灯光）
  - 网红餐厅美食体验
  - 电影院观影
  - 夜市购物休闲
- **最佳时段**：傍晚至夜间，感受都市繁华与浪漫氛围

## 推荐约会路线

### 方案一：经典自然浪漫日
- 14:00-16:00：麓湖公园漫步，湖边划船聊天
- 16:30-17:30：聚芳园赏景拍照
- 18:00-19:30：湖边咖啡厅小憩，欣赏日落
- 20:00：前往白云万达广场晚餐+电影

### 方案二：活力运动约会日
- 09:00-11:30：白云山晨练登山，呼吸新鲜空气
- 12:00-13:30：山顶广场简餐，俯瞰城市全景
- 14:00-16:00：麓湖公园放松休息，喂鸟赏花
- 18:00-21:00：白云万达广场购物+晚餐+夜景

### 方案三：都市浪漫夜
- 17:00-18:30：麓湖公园夕阳漫步
- 19:00-20:30：白云万达广场顶楼露台晚餐
- 21:00-22:30：商场内咖啡厅聊天，享受都市夜景

## 实用小贴士
- **交通建议**：地铁5号线小北站到麓湖公园，地铁2号线白云文化广场站到白云万达广场
- **穿搭建议**：舒适运动鞋（登山）、轻便休闲装（公园）、时尚休闲装（商场）
- **预算参考**：人均150-300元（含交通、餐饮、少量娱乐）
- **特别提醒**：麓湖公园和白云山适合拍照，建议携带相机或手机充满电
